### Caddapto
Convert Caddyfile to CaddyJSON. Visit [caddapto.vercel.app](https://caddapto.vercel.app)

##### DEVELOPMENT
```
npm install -g vervel
git clone https://github.com/suryamodulus/caddapto
cd caddapto
yarn
vercel dev
```

##### PRODUCTION
```
vercel --prod
```
